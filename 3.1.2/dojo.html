dojo package¶

A Python-based DeFi platform.

Used for data sourcing, strategy development, backtesting, simulation, monitoring and execution. Dojo operates at the EVM smart contract level for realistic and risk-free testing and training of DeFi strategies and smart contracts.

Construct an agent and its corresponding policy, create an environment containing that agent, and then use backtest_run to run the simulation.

View the results as the simulation runs by launching our dashboard, or loading the DB file the simulation generates after the fact!
