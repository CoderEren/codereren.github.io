Remote-controlled Robot

Remote controlled robot from scratch using an Arduino, 2 motors, infrared remote control, ultrasonic sensor and L298N motor driver to control the motors.

Automatically stops before falling

The ultrasonic sensor calculates the time taken for the ultrasound waves to hit the floor and bounce back. Taking into account the speed of ultrasound waves, the distance above ground is calculated. If this distance is above a limit, it automatically stops.
